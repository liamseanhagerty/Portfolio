Wii Pong ReadMe

To play with a Wii Remote you will need:
A Nintendo Wii Remote
A compatible Bluetooth Stack

Before starting the application:
1) Pair the Wii Remote with your computer
	-Start up your Bluetooth Software
	-Press and Hold the "1" and "2' Buttons on the Wii Remote Simultaneously
	-Use your Bluetooth Software to search for new devices
	-The Wii Remote is named "Nintendo RVL-CNT-01"
	-If prompted for a Passkey, select "Do Not Use"
	-If prompted for a Service Type, select "keyboard/mouse/HID service"
	-Once the Wii Remote is installed, release the "1" and "2" buttons

2) Disconnect any attachments connected to the Wii Remote
